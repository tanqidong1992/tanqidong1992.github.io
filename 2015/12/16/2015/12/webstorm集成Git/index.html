Webstorm集成Git

Webstorm集成Git

备注

  • 在使用文档过程中，有任何疑问，体会，大家一起讨论学习
  • 阅读文档时，发现任何问题，欢迎修改完善此文档

安装，配置Git

  • 下载Git（Git for windows 不是webStorm插件），然后安装
  • 打开webStorm,打开Git设置 File->settings->Git
  • 修改 Path to Git executable右边输入框的值，改成你安装Git的路径
  • 点击Test按钮，显示出Git的版本信息表示已设置成功，关闭窗口

通过Git导入项目

  • VCS->Checkout from Version Control->Git
  • 输入Git Repository URL（例如：http://tqd@192.168.0.124:8888/r/HNFrontEnd.git）
  • 输入Parent Directory，这个目录将保存你的代码，最好修改一下
  • 项目目录名，不推荐修改，
  • 点击Clone，会提示输入密码，请输入你的Git账号密码

更新本地代码仓库

  • VCS->Update Project
  • 在对话框Update Project中，点击Ok

提交修改

  • 在提交代码之前，切记更新本地代码仓库
  • VCS-> Commit Changes
  • 在Commit Changes对话框中，选中你修改的代码文件，输入Commit Message
  • 在Commit Changes对话框中，点击Commit完成提交到本地代码仓库，Commit and Push提交到本地及远程代码仓库，团队一般使用Commit and Push
  • 在Push Commits对话框中，点击Push，代码修改提交到远程仓库
